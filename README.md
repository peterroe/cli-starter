## ts-starter

A typescript template

## Try it now!

[Create repo from this template on Github](https://github.com/peterroe/ts-starter/generate)

Or:

```bash
$ npx degit peterroe/ts-starter my-ts
$ cd my-ts
$ pnpm i    # npm install -g pnpm
```

## Development

```shell
$ pnpm start
```

## Test

```bash
$ pnpm test
```

## Build

```shell
$ pnpm build
```

## Publish

```shell
$ pnpm release
```
